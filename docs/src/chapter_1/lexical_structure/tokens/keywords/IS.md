# Is, `is`

The `is` keyword is used to **check** if a **value** is of a **specific type**.

## Regex

```regex
is
| es
| est
| lst
| é
| è
| er
| är
| on
| является
| は
| 是
| ~이다
| ni
```

## Examples

```leaf
extern fn foo() {
    char *greeting = "Hello, world!";
    printf("%s", greeting);
}

fn main() {
    foo()
}
```
